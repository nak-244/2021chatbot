Service introduction

ヒューマンリソーシズ事業

詳しくはこちら

当社の人材サービス"Human Resources(ヒューマンリソーシズ)"は企業の雇用外注化ニーズに対応し、よりフレキシブルで確かな品質の提供を実現した人材サービスです。

人材派遣

「必要な人材を必要な時に」
簡単な作業から専門的な業務まで。1日から長期まで。必要なスキルを持った人材を必要な期間だけ、細かなご要望に合わせて幅広くご利用いただけます。

紹介予定派遣

直接雇用を前提とした一定期間の派遣を行い、期間後に双方の合意によって雇用契約を結ぶサービスです。派遣期間を通して見極められるため、ミスマッチを防ぎ効率的な人材確保にご活用いただけます。

人材紹介

採用希望に合った人材を企業へ紹介するサービスです。募集コストを抑え、ニーズに沿った人材をご紹介致します。

海外人材採用

日本の大学・日本語学校で勉強している留学生や、海外の大学等で勉強している学生や現地日系企業に勤めている人材を紹介するサービスです。

BPO事業

詳しくはこちら

営業支援、販売促進、各種サンプリング調査等のアウトバウンドや、通信販売の受信業務など幅広い分野でサービスを提供しております。

営業支援・販売促進・各種サンプリング調査等のアウトバウンドや、テクニカルサポートデスク・通信販売の受信業務など、幅広い分野でのサービスをご提供致します。またダイレクトメール送付や資料送付など【テレマーケティング】～【配送】～【督促】の一環したフルフィルメントサービスの提供まで行っております。

教育事業やITセキュリティ事業としての特徴も持っており、採用から研修を通して高スキルのコミュニケーター/SVを育成。365日対応のサポ－ト体制・プライバシーマークに基づいた万全のセキュリティ体制のもと、お客様の事業体制にマッチしたコンサルティング提案をさせて頂きます。

ITセキュリティ事業

詳しくはこちら

情報の”漏洩、改ざん、なりますまし”を防止するITセキュリティ技術及び、ITセキュリティ技術力を生かしたセキュリティ診断、コンサルティングサービスを提供しております。

ユーザ認証、データ暗号化、デジタル署名関連先端技術を駆使して、無線LAN機器、ネットワーク機器、デジタル家電製品、モバイル機器、ウェブシステム等において、優れたセキュリティソリューションを提供致します。また、IT技術力を活かしたセキュリティ診断、コンサルティングサービス等もお役立ていただいております。
